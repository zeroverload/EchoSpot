# 🌟 项目简介

**EchoSpot（回响坐标）——城市共享设备智能调度平台**

EchoSpot 是一个面向共享充电宝/雨伞等设备场景的高并发本地服务平台。系统基于 **Spring Boot + Redis + RabbitMQ** 构建，围绕“设备查询、租借/归还、智能调度”三大核心流程，解决高并发下设备库存竞争、状态一致性及热点区域资源分布不均等问题。

项目重点在于高性能与高可用架构设计，通过对 Redis 的深度应用，构建了涵盖用户认证、设备租赁、LBS位置服务及社交互动的完整业务闭环。

---

# 🚀 核心亮点

* **高性能架构**：基于 Redis 多级缓存（Caffeine+Redis）显著降低数据库压力，应对高并发租借请求。
* **分布式一致性**：结合 Redis Lua + 分布式锁 + 乐观锁机制，保障“库存扣减 + 一人一设备”的强一致性。
* **异步解耦**：引入 RabbitMQ 实现租借/归还流程异步化，提升系统吞吐量与响应速度。
* **LBS 地理服务**：基于 Redis GEO 实现“附近设备/站点”毫秒级检索与动态排序。
* **设备调度能力**：针对热点区域供需不均问题，支持设备分布优化与调度扩展。

---

# 🏗️ 技术架构

* **核心框架**：Spring Boot + MyBatis-Plus
* **中间件**：Redis + Nginx + RabbitMQ
* **数据存储**：MySQL + Redis
* **工具库**：Hutool + Redisson

# 🔧 技术攻坚与实现细节

## 1. 分布式会话与鉴权体系

* 基于 Redis 存储 Token，实现多节点登录态共享
* 自定义拦截器完成登录校验与 Token 自动续期

---

## 2. 多级缓存与一致性方案

* **缓存策略**：

    * Cache Aside Pattern + 空值缓存 → 解决缓存穿透
    * 逻辑过期 → 防止热点缓存击穿
    * 随机 TTL → 避免缓存雪崩
* **一致性保障**：通过 Redis Lua 脚本保证缓存与业务操作原子性

---

## 3. 共享设备租赁系统（高并发核心）

* **原子校验**：Redis + Lua 一次性完成
  👉 库存校验 + 一人一设备 + 扣减库存 + 写入租借状态
* **分布式锁**：Redisson 保障异步落库阶段幂等性（看门狗续期）
* **异步优化**：

    * 同步阶段：仅完成 Redis 原子操作 + 投递 MQ
    * 异步阶段：由 RabbitMQ 消费端完成订单落库与状态推进

---

## 4. 设备调度与库存管理

* 基于 Redis 缓存设备与站点库存信息，减少数据库访问压力
* 使用 CAS 乐观锁控制库存更新，避免并发冲突
* 支持设备状态管理（空闲/使用中/故障），保证数据一致性

---

## 5. 社交与 LBS 功能

* **点赞排行**：Redis ZSet 实现热度排序与 TopN 查询
* **社交关系**：Set 实现关注/共同关注（交集计算）
* **附近设备**：Redis GEO 实现站点位置存储与距离排序检索
